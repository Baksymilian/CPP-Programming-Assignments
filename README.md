# CPP-Programming-Assignments


This repository contains academic projects developed during the **Object-Oriented Programming Paradigms** course at **Wrocław University of Science and Technology**. The assignments focus on core C++ concepts, design patterns, and efficient code architecture.

## Repository Structure

*   **`01-Polymorphism-Enemy-System`**: A fantasy game enemy system demonstrating dynamic polymorphism, abstract classes, and virtual memory management.
*   **`02-Generic-Templates-Tool`**: A generic mathematical utility for spatial calculations using C++ class and function templates.

## Tech Stack
*   **Language**: C++17
*   **Compiler**: GCC / G++
*   **Tools**: Git, VS Code
*   **Key Concepts**: OOP, Templates, STL, Memory Management
